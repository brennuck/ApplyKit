ApplyKit Add to Chrome

Privacy Policy

Last updated: March 24, 2026

The short version

ApplyKit does not collect, transmit, or share any of your data. Everything stays in your browser. We have no servers, no analytics, no tracking, and no accounts.

What data ApplyKit stores

ApplyKit stores only the information you explicitly enter in the extension's settings page:

  • First name and last name
  • Email address
  • Phone number
  • LinkedIn URL
  • GitHub URL
  • Portfolio URL
  • Custom field labels and values
  • Your uploaded resume file
  • Your preference settings (auto-paste, auto-fill toggles)

Where data is stored

All data is stored locally in your browser using the Chrome Storage API:

  • chrome.storage.sync — your settings (name, email, phone, URLs, custom fields, preferences). This syncs across your Chrome instances if you are signed into Chrome, but is never sent to ApplyKit or any third party.
  • chrome.storage.local — your uploaded resume file. This stays on the device where you uploaded it.

What data ApplyKit does NOT do

  • Does not collect analytics or usage data
  • Does not transmit data to any server
  • Does not use cookies or tracking pixels
  • Does not run any remote or third-party code
  • Does not read or access any page content beyond detecting input fields for auto-fill
  • Does not require an account or login
  • Does not share data with any third party

Permissions used

  • clipboardWrite — to copy your saved values to the clipboard when you click a button
  • storage — to save and retrieve your settings using Chrome's built-in storage

Content scripts

ApplyKit injects a content script on supported job application websites. This script:

  • Displays floating copy buttons on the page
  • Reads input field attributes (name, id, placeholder, labels) to match them with your saved data for auto-fill
  • Does not read or capture any values you type into forms
  • Does not send any page data anywhere

Data deletion

You can delete all your data at any time by clearing the fields in the ApplyKit settings page and clicking Save, or by uninstalling the extension.

Changes to this policy

If this policy is updated, the "Last updated" date at the top will change. Since ApplyKit has no way to contact users (we don't have your email for this purpose), please check this page periodically.

Contact

If you have questions about this privacy policy, you can open an issue on the GitHub repository.
